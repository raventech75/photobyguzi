# Studio Photo - Configurateur à la Carte

Un configurateur de séances photo intelligent built avec Next.js 14 et Tailwind CSS.

## Fonctionnalités

- ✨ Configurateur à la carte intelligent
- 💰 Calcul automatique des prix avec dégressif
- 📱 Design responsive
- 🎨 Interface moderne avec Tailwind CSS
- ⚡ Performance optimisée avec Next.js 14

## Installation

```bash
npm install
npm run dev
